
# Java Character Converter

### char to int 

```java
char c = '1';
int num = Character.getNumbericValue(c);
```

### int to char
```java
int a = 65;  
char c = (char)a;  
```

### char to String
```java
char c = "C";
String s = String.ValueOf(c);

```

### String to char

```java
String s = "hello";  
char c = s.charAt(0); //returns h  
```
